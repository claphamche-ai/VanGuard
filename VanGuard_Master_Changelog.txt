VANGUARD MASTER CHANGELOG
-------------------------
v4.6 - Auto-Healer & Reset Mechanics (Current Build)
- Built `CoreDB.getTenants()` Auto-Healer logic. The system now inspects local storage on load; if legacy arrays are missing new tenant default profiles (e.g., City B), it automatically injects them to prevent ghost desyncs.
- Built a patch protocol to repair `undefined` status fields in legacy tenant data memory, preventing lockout errors on older machines.
- Added a "Factory Reset System" (nukeDatabase) button to the God Dashboard -> Database tab. Wipes local memory and reinstalls factory code.
